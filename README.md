# Emissor de Nota Fiscal (Python + PostgreSQL)

Sistema simples para cadastro de cliente, serviço, geração de nota fiscal e exportação em XML.

## Tecnologias
- Python 3.10+
- PostgreSQL (hospedado em EC2)
- SQLAlchemy ORM
- XML com ElementTree
- Testes com pytest
